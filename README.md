🛡️ IDS Threat Detection System
An interactive Machine Learning-based Intrusion Detection System (IDS) dashboard that monitors network traffic in real time, detects malicious activities, and visualizes security insights using an intuitive web interface.

📌 Overview
The IDS Threat Detection System is a web-based dashboard that simulates real-time network monitoring. It analyzes network packets, classifies traffic using Machine Learning models, and displays live threat alerts through interactive charts and statistics.

This project demonstrates how modern Intrusion Detection Systems combine machine learning algorithms with real-time visualization to improve cybersecurity monitoring.

🧠 Machine Learning Models
The dashboard simulates predictions from multiple ML algorithms:

Random Forest
Neural Network
Support Vector Machine (SVM)
Isolation Forest
Decision Strategy
The system uses:

Majority Voting Ensemble

An alert is generated when at least 3 out of 4 models agree that the traffic is malicious.
